# sample_app

このアプリはハウスメーカーのレビューサイトです。

人生で一回の買い物になるため、家を買うことに対してとても不安があると思いハウスメーカーのレビューを見ること、そして情報を共有できるようなWebアプリがあれば不安を解消できるのではないかと考え、作成しました。

# 機能

* 新規登録、ログイン
* 投稿、画像投稿
* フォロー
* コメント
* お気に入り
* レビュー

# Gem

* ログイン機能        (devise)
* DB                  (MYSQL)
* 検索機能            (ransack)
* ページネーション    (kaminari)
* CSSフレームワーク   (bulma-rails)
